# Installation

## Requirements

- Python 3.9 or higher
- pip

## Install from PyPI

```bash
pip install vald8
```

## Install from Source

```bash
git clone https://github.com/osuthorpe/vald8.git
cd vald8
pip install -e .
```

## Verify Installation

```python
import vald8
print(vald8.__version__)
```

## Optional Dependencies

For development:

```bash
pip install vald8[dev]
```

For documentation:

```bash
pip install vald8[docs]
```

For everything:

```bash
pip install vald8[all]
```
